# Analysis of *Sarcotaces* sp. infections in BC groundfish

Analysis code and data for the paper:

Siegle, M.R., Dunic, J.C., Castle, K., Wyeth, M., Love, M.S.,
Anderson, S.C. Prevalence of the parasitic copepod, *Sarcotaces* sp., infection in British Columbia rockfishes (*Sebastes* spp.) and implications for rockfish life-history. In press at the Canadian Journal of Fisheries and Aquatic Sciences.

## Main folders

- `R/`: primary analysis scripts
- `data-raw/`: raw lookup tables and source specimen data used for cleaning
- `data-generated/`: cleaned datasets, intermediate objects, and saved model outputs (generated by scripts)
- `figures/`: main figures used for the manuscript (generated by scripts)

## Script order

The main workflow runs through the numbered scripts in `R/`:

1. `R/00-prep-data.R`: clean and subset source data; write `.rds` files to `data-generated/`
2. `R/01-summary-results.R`: create descriptive summaries and some manuscript figures
3. `R/01-map.R`: fit the spatial model and produce the map/depth-related outputs
4. `R/02-maturity-length.R`, `R/02-maturity-age.R`, `R/02-maturity-bins.R`, `R/02-body-condition.R`, `R/02-age-infection.R`: fit the main biological models and generate figures
5. `R/03-make-tables.R`: generate manuscript tables
6. `R/get-values.R`: write LaTeX macros to `text/values.tex` for manuscript text
